Ejercicio I

El comando adduser se utiliza para crear nuevas cuentas de usuario de manera interactiva. Su utilización es sencilla. Sólo es necesario, como mínimo, especificar el nombre de usuario.

Ej: adduser jgonzalez

El sistema luego, le pedirá, de manera interactiva, información extra de la nueva cuenta a crear (el nombre completo real, la clave inicial, etc).

Ejercicio V
-----------

El comando addgroup funciona de manera similar a adduser pero se utiliza para crear nuevos grupos de usuario. Ejemplo de uso: addgroup empleados (crea en el sistema un grupo llamado empleados).

Utilice addgroup para crear tres nuevos grupos en el sistema:

	- directorio
	- empleados
	- todos


Ejercicio VI
------------

El comando usermod puede ser útil para agregar usuarios a grupos, o modificar algunas opciones del usuario. Por ejemplo, para agregar el usuario "director" al grupo "todos", puede ejecutar usermod -G todos -a director.
Verifique con el manual que hacen las opciones -G -a para el comando usermod.
Modifique los grupos utilizando usermod para agregar usuarios a los grupos. De la siguiente manera:

	- El grupo directorio debería estar compuesto por los usuarios director, legales y contaduría.

	- El grupo empleados debería estar compuesto por secretaria, contaduría, legales y buffet.

	- El grupo todos debería estar compuesto por director, legales y contaduría
